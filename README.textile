h1. Greasemonkey- / user-scripts for client-side UI/UX customization of Redmine


h2. Description

This is a collection of scripts to improve look and feel of Redmine project management, client-sided locally in your browser.

These scripts can be used with the greasemonkey plugin in firefox and as user scripts in Chrome (we recommend administration via the Tampermonkey extension).



h2. Installation Notes

Ensure to update the @match setting for the URL of your Redmine installation when adding a script to your browser.
